# Repositório foi criado para realizar o desafio da Unidac
## Tecnologias usadas:
Arquitetura de software com o backend e frontend devidamente separados:

Back-End:
- Spring boot;
- Postman para a realização de testes da API.

Front-End:
- ReactJs;
- Bootstrap.
